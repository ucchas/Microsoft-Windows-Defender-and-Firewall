Configuring and Updating Windows Defender

OS: Windows 10

Objective:

  • Review Windows security virus and threat protection
  • Update threat definitions
  • Run Windows Defender antivirus quick scan

Review Windows Security Virus & threat protection

Settings > Update & Security > Windows Security > Virus & threat protection

  • Current threats: Shows any threats that have been detected on the device. Contains information such as when the last scan occurred, how long the scan took, and how many flies were scanned. There is also scan options which lets us run a full scan, a quick scan or a custom scan.

  • Virus & threat protection settings: Using this we can customize the protection level, opt to send sample files to Microsoft, exclude files or folders from scan, or temporarily stop the protection.

  • Virus & threat protection updates: To view the last time the virus definitions were updated. We can also opt to manually check for updates.
  • Ransomware protection: To choose to enable controlled folder access. This protects memory, files, and folders from unauthorized changes.

Update Threat Definitions

Under Virus & threat protection updates > Check for updates

Here, we can see the definition version and the associated date and time when the last update was performed. Now selected Check for updates.

After it is done updating, the Check for updates button will reappear. The Threat definition version has been changed from 1.407.20.0 to 1.407.24.0. The date and time associated with last performed definition update has also changed.

Run Antivirus Quick Scan

Under Virus & threat protection screen there is an option to run a Quick scan.

After it is done, the Quick scan button will reappear. The recent findings can be seen by clicking on the Threat history button.

This page shows the results of the scan. It shows the files identified as a threat and quarantined along with files identified as potential threats but allowed to continue running.

Custom scan on a folder

Objective:

  • Run a custom scan that only scans the files in the Downloads folder

Under Virus & threat protection:

Scan options > Custom scan > Scan now > select folder path

Here, we can see the virus scan results for the Downloads folder.
